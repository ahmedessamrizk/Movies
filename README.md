# 🎬 Movies App

Movie browsing app using TMDB API.

🔗 Live Demo: https://ahmedessamrizk.github.io/Movies/

## 🚀 Features
- Browse trending & categories  
- Search movies  
- Animated sidebar  
- Form validation  

## 🛠️ Tech
HTML, CSS, JavaScript, jQuery, Bootstrap, TMDB API

## 📌 Notes
- Uses Fetch API  
- Regex validation  
- API key exposed (not for production)
